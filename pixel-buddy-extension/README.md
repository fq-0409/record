# PixelBuddy Usage Guide

## Installation
1. Open Chrome and go to `chrome://extensions/`
2. Enable "Developer mode" in the top right corner.
3. Click "Load unpacked".
4. Select the `pixel-buddy-extension` folder in this directory.

## Features
- **Focus Timer**: Click "Start Focus" to begin a 25-minute Pomodoro session.
- **Pixel Pet**: Your pet reacts to your health status.
- **Monitoring**: If you visit a blacklisted site (YouTube, Bilibili, etc.) while focusing, you lose HP!
- **Leveling**: Complete sessions to gain XP and Level Up.
- **Options**: Right-click the extension icon and select "Options" to manage your blacklist.

## Troubleshooting
- If the timer doesn't seem to work, try reloading the extension.
- Make sure notifications are enabled for Chrome.
