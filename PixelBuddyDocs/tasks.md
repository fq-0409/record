## 任务清单（tasks.md）

**开发计划**：

第一批功能（MVP 验证）
- 初始化 Chrome Extension V3 项目结构（编写 manifest.json）
- 使用 Tailwind 搭建基础 Popup 弹窗页面（倒计时 + 血条展示）
- 实现并跑通 25 分钟番茄钟计时器逻辑

第二批功能（核心规则）
- 编写 Background Script 监控浏览器标签页变化
- 实现网站黑名单比对逻辑和“扣血”惩罚机制
- 编写并集成 `chrome.storage` 记录等级和经验数据

第三批功能（视觉完善）
- 引入或通过 AI 生成像素小人资产（例如正常/受伤/开心的状态图片）
- 添加血条变化的绿色/红色 UI 动画反馈
- 修复边界 Bug（如计时期间关闭浏览器的处理）

编译阶段
只关注函数的声明，检查类型匹配（可做隐式类型转换）、翻译函数名
链接阶段
在函数的定义中（.o文件）进行修饰的函数名匹配
